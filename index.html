    Global Anthro Notes Installation:

    1. Download Latest GAN zip
    2. Extract to your "My Paratext Projects" folder
    3. In Paratext, File > Open File/Resource... to a scripture project or resource
    4. View > Show Consultant Notes > GAN (Global Anthro Notes)
    5. Navigate to Luke 1:1 in the scripture project or resource
